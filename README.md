## 機能

* 指定した時間(分)経過したら、音声で通知する
* MacOSの`say`コマンドを使用しているため、MacOSのみ対応

## 実行例

* コマンド引数に数字を入力する
* 入力した順に経過時間(分)

```bash
# 22分タイマー:12分経過時に音声通知をする
$ node bin/index.js 12 10
```

## コマンド登録

* プロジェクトルートディレクトリに移動

```bash
$ npm install -g
```

## コマンド解除

* プロジェクトルートディレクトリに移動

```bash
$ npm uninstall -g
```
